ATOR: Administrador

INCLUDE: Login Usuario

FLUXO BASICO:

1 - O Sistema apresenta a 'Tela de gerenciamento', onde o administrador seleciona a opcao 'cadastrar Agente'.
2 - O sistema aprensenta os campos necessarios para cadastramento das informacoes do agente.(alt.1)(info.1)
3- O administrador preenche os campos necessarios e enviar a solicitacao para ser processadas. (alt.2)
4 - O sistema confirma a realizacao do cadastramento.(msg.3)
5 - O sistema retorna para a 'Tela de Gerenciamento'.



FLUXO ALTERNATIVO:

alt.1 - Erro na Pagina: 
	1 - O sistema encaminha o usuario para a pagina de gerenciamento e exibe a mensagem de erro. (msg.1)

alt.1 - Login o senha Incorreto/inexistente
	1 - O sistema encaminha a solicitacao para a pagina de login com a seguinte mensagem msg.1

alt.2 - Erro ao cadasatrar e/ou servidor inacessivel
	1 - O sistema devera encaminhar para tela principal com a seguinte mensagem msg.2


MENSAGENS:

msg.1 - Erro no processamento, pagina solicitada não pôde ser acessada.
msg.1 - Usuario nao cadastrado ou email ou senha incorretos.
msg.2 - servico indisponivel ou dados cadastrados de forma incorreta.
msg.3 - Conta de Agente cadastrada com sucesso! 

CAMPOS

info.1 - 'Nome do Agente'. 'endereco', 'email', 'senha'.

FLUXO ALTERNATIVO:

alt.1 - Erro na Pagina: 
	1 - O sistema encaminha o usuario para a pagina principal e exibe a mensagem de erro. (msg.1)

alt.2 - Dados preenchidos inválidos:
	1 - O sistema identifica os campos e os destaca para o usuario. (msg.2)


MENSAGENS:

msg.1 - Erro no processamento, pagina solicitada não pôde ser acessada.
msg.2 - campos com preenchimento vazio/invalido.
msg.3 - Cadastro Realizado com Sucesso!
